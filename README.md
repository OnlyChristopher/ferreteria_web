# Ferretería — E-commerce Web

Proyecto front-end de una tienda de ferretería pensado como prototipo y base para un e-commerce. Incluye diseño de interfaz y configuración para desarrollo local.

## Descripción
Aplicación web de ejemplo para gestionar catálogo y navegación de productos de ferretería. Está pensada como plantilla para aprender y extender con funcionalidades reales (carrito, autenticación, pasarela de pago, etc.).

## Características
- Interfaz responsiva para escritorio y móvil.
- Componentes reutilizables para catálogo, filtros y producto.
- Flujo básico de navegación y vistas de producto.
- Preparado para desarrollo local y despliegue.

## Requisitos
- Node.js (versión LTS recomendada)
- npm o yarn

## Instalación y ejecución
1. Instalar dependencias:
   npm install

2. Iniciar servidor de desarrollo:
   npm run dev

3. Construir para producción:
   npm run build

4. Ejecutar build en modo producción:
   npm run start

(Asegúrate de revisar los scripts en package.json para comandos adicionales.)

## Scripts útiles
- npm run dev — servidor de desarrollo con recarga en caliente.
- npm run build — compila la aplicación para producción.
- npm run start — ejecuta la versión compilada (producción).
- npm test — ejecutar pruebas (si están configuradas).
